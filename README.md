# MCMessage

## Usage

#### For Maven:

```xml
<dependency>
    <groupId>com.github.siroshun09.mcmessage</groupId>
    <artifactId>mcmessage</artifactId>
    <version>2.0.0</version>
</dependency>
```

#### For Gradle:

```
    dependencies {
        compile 'com.github.siroshun09.mcmessage:mcmessage:2.0.0'
    }
```
