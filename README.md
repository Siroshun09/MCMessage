# MCMessage

## Usage

#### For Maven:

```xml
    <repository>
        <id>jcenter</id>
        <url>https://jcenter.bintray.com</url>
    </repository>
```

```xml
    <dependency>
        <groupId>com.github.siroshun09.mcmessage</groupId>
        <artifactId>mcmessage</artifactId>
        <version>1.0.0</version>
    </dependency>
```

#### For Gradle:

```
    repositories {
        jcenter()
    }
```

```
    dependencies {
        compile 'com.github.siroshun09.mcmessage:mcmessage:1.0.0'
    }
```
